建表命令行：
hbase shell
create 'test',{NAME => 'data'}

建hdfs文件夹：hadoop fs -mkdir /datas

把文件放到hdfs文件夹：hadoop fs -put *.csv /datas

导数据：
hbase org.apache.hadoop.hbase.mapreduce.ImportTsv -Dimporttsv.bulk.output=output1 -Dimporttsv.columns=HBASE_ROW_KEY,data:post test /datas

hbase org.apache.hadoop.hbase.mapreduce.LoadIncrementalHFiles output1 test

装AWS：
$ curl "https://s3.amazonaws.com/aws-cli/awscli-bundle.zip" -o "awscli-bundle.zip"
$ sudo apt-get install unzip
$ unzip awscli-bundle.zip
$ sudo ./awscli-bundle/install -i /usr/local/aws -b /usr/local/bin/aws

install java 8 
sudo add-apt-repository ppa:webupd8team/java 
sudo apt-get update 
sudo apt-get install oracle-java8-installer
